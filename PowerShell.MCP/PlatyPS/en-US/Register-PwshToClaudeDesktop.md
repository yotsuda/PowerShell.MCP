# Register-PwshToClaudeDesktop

## SYNOPSIS

Registers PowerShell.MCP as an MCP server in Claude Desktop.

## SYNTAX

### __AllParameterSets

```
Register-PwshToClaudeDesktop [<CommonParameters>]
```

## ALIASES

This cmdlet has no aliases.

## DESCRIPTION

Adds or updates the 'pwsh' entry in Claude Desktop's claude_desktop_config.json. Existing settings in the file are preserved.

## EXAMPLES

### Example 1

Registers PowerShell.MCP in Claude Desktop configuration.
